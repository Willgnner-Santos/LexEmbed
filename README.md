# LexEmbed
Article reproducibility LexIris-pt and LexBert-pt: Specialized Sentence Embeddings for Legal Similarity in Brazilian Portuguese.

# How to use
Each folder contains the code, as well as documentation on how to use it.

# Corpora
The data used in this study are available by authorization of the granting institution, with the authors acting as intermediaries. To request access, simply send an email to the authors (contact details are provided in the article) indicating your interest and the intended use of the data.
